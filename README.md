# AI Infra Simulator

A standalone drag-and-drop simulator for designing AI system infrastructure and estimating real costs.

**[→ Open the simulator](https://anaishadh.github.io/ai-infra-simulator/)**

## Features
- 43 components across 6 categories (LLM, Memory, Agent, Pipeline, Infra, Observability)
- Real 2025 market pricing (OpenAI, AWS, Pinecone, Datadog, etc.)
- Infinite canvas with zoom and pan
- Directed connections with custom routing
- 8 industry-accurate templates
- Build validator with connection direction checks
- Export architecture spec as text

## Usage
No install needed. Open the link above in any modern browser.
To run locally: download `index.html` and open it — works fully offline.
